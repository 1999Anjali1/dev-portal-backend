# 🔧 DevPortal API — NestJS Backend

REST API backend for the DevPortal application built with NestJS, TypeORM and PostgreSQL.

## 🔗 Related Repositories

- **Frontend**: [dev-portal-frontend](https://github.com/1999Anjali1/dev-portal-frontend) — Angular 21 Frontend

## 🛠️ Tech Stack

| Technology | Version | Purpose |
|---|---|---|
| NestJS | 11.x | Backend Framework |
| TypeScript | 5.x | Programming Language |
| TypeORM | - | Database ORM |
| PostgreSQL | 17.x | Database |

## 📡 API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| GET | /developers | Get all developers |
| GET | /developers/:id | Get one developer |
| POST | /developers | Create developer |
| DELETE | /developers/:id | Delete developer |

## 🚀 Getting Started

```bash
git clone https://github.com/1999Anjali1/dev-portal-backend.git
cd dev-portal-backend
npm install
npm run start:dev
```

API runs at `http://localhost:3000`

## ⚙️ Environment Setup

Make sure PostgreSQL is running with:
- Database: `devportal`
- Username: `postgres`
- Port: `5432`
